What is an Electric Vehicle?

The electric vehicle is a vehicle that runs on electricity alone.
Such a vehicle does not contain an internal combustion engine like the other conventional vehicles.Instead, it employs an electric motor to run the wheels.
These vehicles are becoming very popular nowadays.They are considered to be a promising solution for the future transportation. The most common example is Tesla.

Main components of an Electric Vehicle:

* Electric Battery
* Inverter
* Electric Motor
* Regenerative Braking System
* Power Control Unit
